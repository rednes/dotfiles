# dotfiles

## My dotfiles for:

* emacs
* fish
* git(ignore global, config)
* screen
* textlint
* tmux
* vim
* zsh

## Install:

```sh
$ git clone https://github.com/rednes/dotfiles.git
$ ./dotfiles/install.sh
```
